Tentang Proyek Ini

Halo! Nama saya Rakhmat Senoaji. Saya adalah seorang mining engineer yang tertarik dengan penerapan GIS karena latarbelakang saya dari Teknik Geodesi. Proyek ini dibuat sebagai bagian dari latihan untuk memahami dasar-dasar HTML dan struktur website sederhana yang termasuk tugas Task Day 8 MAPID WebGIS Bootcamp Batch 2. Tujuan website ini adalah untuk memberikan informasi tentang jalur pendakian di Bandung, sehingga para pendaki dapat merencanakan perjalanan mereka dengan lebih baik.

Pendidikan

Saya saat ini sedang mempelajari pengembangan web dan sistem informasi geografis melalui bootcamp WebGIS. Proyek ini merupakan bagian dari tugas latihan untuk membuat website pariwisata sederhana hanya menggunakan HTML.

  1. Mempelajari struktur dasar HTML.
  2. Melatih penggunaan tags seperti navbar, main, dan footer.
  3. Mempersiapkan pengembangan peta WebGIS pada tahap berikutnya.

Latar Belakang

Bandung memiliki banyak destinasi pendakian yang menarik, mulai dari jalur hutan hingga titik pandang pegunungan. Proyek ini dibuat untuk mendukung informasi pariwisata dan membantu pengguna membandingkan detail jalur sebelum merencanakan perjalanan.

Ke depannya, website ini dapat menampilkan informasi rute yang lebih lengkap, termasuk akses transportasi, perkiraan waktu pendakian, kenaikan elevasi, dan tingkat kesulitan jalur.
